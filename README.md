# Yandex Archive Downloader (Chrome Extension)

Расширение скачивает страницы открытого документа Яндекс.Архив в один файл:
- `ZIP` (набор изображений страниц),
- `PDF` (единый документ).

## Установка (для всех пользователей)

1. Скачайте или клонируйте проект на компьютер.
2. Откройте Chrome и перейдите на `chrome://extensions`.
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку проекта `yandexarchiver`.

## Как пользоваться

1. Откройте нужный документ в Яндекс.Архив.
2. Нажмите на иконку расширения `Yandex Archive Downloader`.
3. Выберите формат: `ZIP` или `PDF`.
4. Нажмите **Начать** и дождитесь завершения.
5. В окне загрузки Chrome выберите, куда сохранить файл.
