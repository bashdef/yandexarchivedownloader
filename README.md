# Yandex Archiver Downloader (Chrome Extension)

Расширение скачивает страницы открытого документа Яндекс.Архив в один файл:
- `ZIP` (набор изображений страниц),
- `PDF` (единый документ).

## Установка (для всех пользователей)

1. Скачайте или клонируйте проект на компьютер.
2. Откройте Chrome и перейдите на `chrome://extensions`.
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку проекта `yandexarchiver`.

## Как пользоваться

1. Откройте нужный документ в Яндекс.Архив (пример: `https://yandex.ru/archive/catalog/...`).
2. Если есть проверка "Я не робот", пройдите ее вручную.
3. Нажмите на иконку расширения.
4. Выберите формат: `ZIP` или `PDF`.
5. Нажмите **Начать** и дождитесь завершения.
6. В окне загрузки Chrome выберите, куда сохранить файл.

## Важно

- Расширение работает с открытым документом во вкладке.
- Если конкретный документ отображается нестандартно, сбор страниц может потребовать доработки селекторов.
