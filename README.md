# Yandex Archive Downloader (Chrome Extension)

Расширение скачивает страницы открытого документа Яндекс.Архив в один `PDF` файл:

## Установка (для всех пользователей)

1. Скачайте архив и распакуйте или клонируйте проект на компьютер.
2. Откройте Chrome и перейдите на `chrome://extensions`.
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите папку с расширением.

## Как пользоваться

1. Откройте нужный документ в Яндекс.Архив.
2. Нажмите на иконку расширения `Yandex Archive Downloader`.
3. Нажмите **Скачать PDF** и дождитесь завершения (вкладку можно закрыть).
4. В окне загрузки Chrome выберите, куда сохранить файл.
